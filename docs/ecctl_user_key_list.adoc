== ecctl user key list

Lists the API keys for the specified user, or all platform users

[float]
=== Synopsis

Lists the API keys for the specified user, or all platform users

----
ecctl user key list --user=<user id>|--all [flags]
----

[float]
=== Options

----
      --all    If specified, lists all API keys for all platform users
  -h, --help   help for list
----
