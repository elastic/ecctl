== ecctl deployment note create

Adds a note to a deployment

[float]
=== Synopsis

Adds a note to a deployment

----
ecctl deployment note create <deployment id> --comment <comment content> [flags]
----

[float]
=== Options

----

  -h, --help             help for create
----
