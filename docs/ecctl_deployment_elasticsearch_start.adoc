== ecctl deployment elasticsearch start

Starts a stopped Elasticsearch cluster

[float]
=== Synopsis

Starts a stopped Elasticsearch cluster

----
ecctl deployment elasticsearch start <cluster id> [flags]
----

[float]
=== Options

----
  -h, --help    help for start
  -t, --track   Tracks the progress of the performed task
----
